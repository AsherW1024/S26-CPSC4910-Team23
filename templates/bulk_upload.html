Bulk User Upload

{% if session.get("role")=="Sponsor" %}

Use the file upload below to upload a .txt file containg the user information you want to upload

It should be formatted like...

type||first name|last name|email address|points|reason for points

The type refers to the user type (driver or sponsor) that the rest of the line is defining the data for

There should be 2 "|" characters between the type and first name

Use the character D to define or add points to a driver account and the character S for sponsor accounts

{% elif session.get("role")=="Admin" %}

Use the file upload below to upload a .txt file containing the records you want to process.

Driver format:

D|organization name|first name|last name|email address|points|reason for points

Organization format:

O|organization name

If a driver email already exists and belongs to a driver account, bulk upload will update that driver's points and auto-accept them into the uploaded organization.

If a driver does not exist yet, bulk upload will create the driver, assign them to the uploaded organization, and mark the uploaded application as accepted automatically.
